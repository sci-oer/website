{{< /blocks/cover >}} {{% pageinfo %}} This site is still under construction. {{% /pageinfo %}} {{% blocks/lead color="primary" %}} This resource is part of an investigation into a semi-automated process for creating Self-Contained, Interactive Open Educational Resources (SCIOER) for use by students and instructors in the discipline of Computer Science. Resources created through the SCIOER process are self-contained andprovide all the tools necessary for learners. The interactive nature of the resource encourages experimentation and incremental practice. Learners are able to annotate SCIOER resources with their personal observations and learnings. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-solid fa-satellite-dish" title="Offline Access" %}} Everything you need is contained within the resource! No internet access required! {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions Welcome!" %}} We do a [Pull Request](https://github.com/sci-oer/student-cli/pulls) contributions workflow on **GitHub**. New users are always welcome! {{% /blocks/feature %}} {{% blocks/feature icon="fa-solid fa-person-chalkboard" title="Create Custom Courses" url="https://github.com/sci-oer/automated-builder" %}} Purpose built to be easily extensible by instructors wishing to create their own open educational resource. {{% /blocks/feature %}} {{% blocks/feature icon="fa-brands fa-docker" title="Docker Images" url="https://hub.docker.com/u/scioer" %}} Based on the industry standard containerization methods using Docker. We provide a suite of prebuilt images that can be used to create a range of easily attributable courses. {{% /blocks/feature %}} {{% blocks/feature icon="fa-solid fa-desktop" title="Cross Platform" %}} Designed to be compatible for Windows, MacOS, and linux! It even works on both Intel/AMD systems _and_ Apple Silicon! {{% /blocks/feature %}} {{% blocks/feature icon="fa-solid fa-face-smile" title="Student Satisfaction" %}} "10 out of 10.
